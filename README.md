## Features
- Creating backup of files and directories with the suffix baka
- Confirmation if the file or directory is larger than 1GiB

## Manual installation

### Dependencies
- make
- dialog

### Download latest release

### Unpack and change directory
```
tar -xvzf baka-?.?.?.tar.gz
cd baka-?.?.?
```
### Install
```
make install
```
